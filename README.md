# Game Edukasi Susun Huruf & Kata Bahasa Inggris

## Deskripsi
Game edukasi berbasis Unity untuk siswa SD yang berfokus pada:
- Menyusun huruf (kosakata)
- Menyusun kata (frasa)

Game ini menggunakan sistem skor, nyawa (buku), dan progres stage
untuk meningkatkan motivasi belajar siswa.

---

- Integrasi backend (web admin & database)

## Teknologi

- REST API
