# Game Edukasi Susun Huruf & Kata Bahasa Inggris

## Deskripsi
Game edukasi berbasis Unity untuk siswa SD yang berfokus pada:
- Menyusun huruf (kosakata)
- Menyusun kata (frasa)
Menggunakan sistem skor, nyawa (buku), dan progres stage.

## Peran Saya
- Game Programmer (Unity C#)
- Gameplay System & UI Logic
- Integrasi backend (web admin & database)

## Teknologi
- Unity (C#)
- PHP & MySQL (Web Admin)
- REST API

## Catatan
Source code web admin tidak dipublikasikan sepenuhnya
karena menggunakan template pihak ketiga dan digunakan
oleh institusi pendidikan.
